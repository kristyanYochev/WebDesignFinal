Overview

You can use our API in order to embed a stream in your apps. You can also use our OAuth authentication system to manage users on the platforms you are developing.

Streaming API

Our streaming API is very easy to use - just import our npm package, call a function and we take care of the rest. You can read more about it here.

OAuth API

We have exposed OAuth API, so that you can manage users on your platform without worrying about passwords and email addresses. All you have to do is grab a token from our API console, write 10 lines of code and you are good to go.
